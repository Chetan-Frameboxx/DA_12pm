## Practice Exercises

1. Write a program to check whether a number is positive, negative, or zero.
2. Write a program to determine whether a person is a child, teenager, or adult based on age.
3. Write a program to check whether a given year is a leap year. (A year is a leap year if divisible by 4, but not by 100, except if also divisible by 400).
4. Write a program to find the largest of three numbers entered by the user.
